# 🌌 Midnight Luxury Weather Dashboard

A premium weather dashboard built using Streamlit and OpenWeather API with a modern glassmorphism UI.

## ✨ Features

- 🌤 Real-time weather data
- 📈 5-day weather forecast
- 🌫 Air Quality Index (AQI)
- 🌍 Interactive location map
- 🌅 Sunrise & Sunset timing
- 📊 Weather analytics charts
- 📥 Download forecast CSV
- 🌌 Premium Midnight Luxury UI

## 🛠 Tech Stack

- Python
- Streamlit
- Plotly
- Pandas
- OpenWeather API

## 📂 Project Structure

```txt
weather-dashboard/
│── app.py
│── requirements.txt
│── README.md
│── assets/
│   └── style.css
```

## 🚀 Installation

Clone repository:

```bash
git clone YOUR_REPO_LINK
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```

## 🔑 API Setup

1. Create an account on OpenWeather
2. Generate API key
3. Paste API key into dashboard sidebar
